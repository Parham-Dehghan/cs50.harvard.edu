درس‌نامه هفته دوم: مبانی برنامه‌نویسی در C

بازگشت به صفحه اصلی

خوش‌آمدگویی!

در جلسه قبلی، ما با زبان برنامه‌نویسی C، یک زبان متنی، آشنا شدیم. این هفته، به بررسی عمیق‌تر بلوک‌های سازنده اضافی می‌پردازیم که به ما در یادگیری بیشتر برنامه‌نویسی از پایه کمک می‌کنند.

به طور اساسی، علاوه بر مبانی برنامه‌نویسی، این دوره درباره حل مسئله است. بنابراین، ما همچنین روی نحوه برخورد با مسائل علوم کامپیوتر تمرکز خواهیم کرد. تا پایان این دوره، شما یاد خواهید گرفت که چگونه از این بلوک‌های سازنده برای حل مجموعه‌ای از مسائل علوم کامپیوتر استفاده کنید.

سطوح خواندن

یکی از مسائل واقعی که در این دوره حل خواهیم کرد، درک سطوح خواندن است. با کمک برخی از همکلاسی‌هایتان، ما متونی با سطوح خواندن مختلف ارائه کردیم. این هفته، ما سطوح خواندن را به عنوان یکی از چالش‌های برنامه‌نویسی شما کمی‌سازی خواهیم کرد.

کامپایل

رمزنگاری عملی است برای مخفی کردن متن ساده از دید افراد غیرمجاز. رمزگشایی، در نتیجه، عملی است برای تبدیل متن رمزگذاری‌شده به شکلی قابل خواندن برای انسان. یک متن رمزگذاری‌شده ممکن است به این شکل باشد:

U  I  J  T   J  T   D  T  5  0

هفته گذشته، شما با کامپایلر آشنا شدید، برنامه کامپیوتری خاصی که کد منبع را به کد ماشین قابل فهم برای کامپیوتر تبدیل می‌کند. به عنوان مثال، ممکن است برنامه‌ای به این شکل داشته باشید:

#include 

int main(void)
{
    printf("hello, world\n");
}

کامپایلر کد بالا را به کد ماشین تبدیل می‌کند. VS Code، محیط برنامه‌نویسی ارائه‌شده برای دانشجویان CS50، از کامپایلری به نام clang یا زبان C استفاده می‌کند. می‌توانید دستور زیر را در پنجره ترمینال وارد کنید تا کد خود را کامپایل کنید: clang -o hello hello.c.

اجرای ./hello در پنجره ترمینال، برنامه شما را به صورت مورد انتظار اجرا می‌کند. کد زیر را از هفته گذشته در نظر بگیرید:

#include 
#include 

int main(void)
{
    string name = get_string("What's your name? ");
    printf("hello, %s\n", name);
}

برای کامپایل این کد، می‌توانید clang -o hello hello.c -lcs50 را تایپ کنید. اگر make hello را تایپ کنید، دستوری اجرا می‌شود که clang را برای ایجاد یک فایل خروجی قابل اجرا توسط کاربر اجرا می‌کند.

کامپایل شامل چهار مرحله اصلی است:

  • پیش‌پردازش: فایل‌های هدر (مانند cs50.h) به کد کپی می‌شوند.
  • کامپایل: کد به کد اسمبلی تبدیل می‌شود.
  • اسمبل: کد اسمبلی به کد ماشین تبدیل می‌شود.
  • لینک: کد از کتابخانه‌ها با کد شما ترکیب می‌شود.

اشکال‌زدایی

همه در هنگام کدنویسی اشتباه می‌کنند. اشکال‌زدایی فرآیند یافتن و حذف باگ‌ها از کد شما است. یکی از تکنیک‌های اشکال‌زدایی که در این دوره استفاده خواهید کرد، اشکال‌زدایی با اردک لاستیکی است، جایی که می‌توانید با یک شیء بی‌جان (یا خودتان) صحبت کنید تا به شما کمک کند درباره کدتان و دلیل کار نکردن آن فکر کنید.

کد زیر را در نظر بگیرید که عمداً یک باگ در آن قرار داده شده است:

#include 

int main(void)
{
    for (int i = 0; i <= 3; i++)
    {
        printf("#\n");
    }
}

این کد چهار بلوک به جای سه بلوک چاپ می‌کند. می‌توانید با افزودن printf برای اشکال‌زدایی کد را اصلاح کنید:

#include 

int main(void)
{
    for (int i = 0; i <= 3; i++)
    {
        printf("i is %i\n", i);
        printf("#\n");
    }
}

اجرای این کد، چندین عبارت را خواهید دید، از جمله i is 0، i is 1، i is 2 و i is 3. با دیدن این، ممکن است متوجه شوید که کد باید به این صورت اصلاح شود:

#include 

int main(void)
{
    for (int i = 0; i < 3; i++)
    {
        printf("#\n");
    }
}

ابزار دیگر اشکال‌زدایی، دیباگر VS Code است. می‌توانید یک نقطه توقف تنظیم کنید و debug50 ./buggy0 را اجرا کنید تا کد را مرحله به مرحله بررسی کنید.

آرایه‌ها

هر نوع داده به مقدار مشخصی از منابع سیستم نیاز دارد:

  • bool: 1 بایت
  • int: 4 بایت
  • long: 8 بایت
  • float: 4 بایت
  • double: 8 بایت
  • char: 1 بایت
  • string: ? بایت

آرایه‌ها دنباله‌ای از مقادیر هستند که به صورت پشت سر هم در حافظه ذخیره می‌شوند. به عنوان مثال:

#include 

int main(void)
{
    int score1 = 72;
    int score2 = 73;
    int score3 = 33;
    printf("Average: %f\n", (score1 + score2 + score3) / 3.0);
}

این کد را می‌توان با استفاده از آرایه بهبود داد:

#include 
#include 

int main(void)
{
    int scores[3];
    scores[0] = 72;
    scores[1] = 73;
    scores[2] = 33;
    printf("Average: %f\n", (scores[0] + scores[1] + scores[2]) / 3.0);
}

می‌توانیم با یک حلقه و تابع میانگین را ساده‌تر کنیم:

#include 
#include 

const int N = 3;
float average(int length, int array[]);

int main(void)
{
    int scores[N];
    for (int i = 0; i < N; i++)
    {
        scores[i] = get_int("Score: ");
    }
    printf("Average: %f\n", average(N, scores));
}

float average(int length, int array[])
{
    int sum = 0;
    for (int i = 0; i < length; i++)
    {
        sum += array[i];
    }
    return sum / (float) length;
}

رشته‌ها

یک string به سادگی یک آرایه از متغیرهای نوع char است. به عنوان مثال:

#include 

int main(void)
{
    char c1 = 'H';
    char c2 = 'I';
    char c3 = '!';
    printf("%c%c%c\n", c1, c2, c3);
}

با جایگزینی %c با %i، کدهای ASCII چاپ می‌شوند:

#include 

int main(void)
{
    char c1 = 'H';
    char c2 = 'I';
    char c3 = '!';
    printf("%i %i %i\n", c1, c2, c3);
}

می‌توانیم رشته‌ها را به صورت آرایه‌ای از کاراکترها بررسی کنیم:

#include 
#include 

int main(void)
{
    string s = "HI!";
    printf("%c%c%c\n", s[0], s[1], s[2]);
}

برای محاسبه طول یک رشته:

#include 
#include 

int main(void)
{
    string name = get_string("Name: ");
    int n = 0;
    while (name[n] != '\0')
    {
        n++;
    }
    printf("%i\n", n);
}

یا با استفاده از کتابخانه string.h:

#include 
#include 
#include 

int main(void)
{
    string name = get_string("Name: ");
    int length = strlen(name);
    printf("%i\n", length);
}

برای تبدیل کاراکترهای کوچک به بزرگ:

#include 
#include 
#include 
#include 

int main(void)
{
    string s = get_string("Before: ");
    printf("After:  ");
    for (int i = 0, n = strlen(s); i < n; i++)
    {
        printf("%c", toupper(s[i]));
    }
    printf("\n");
}

آرگومان‌های خط فرمان

آرگومان‌های خط فرمان به برنامه منتقل می‌شوند:

#include 
#include 

int main(int argc, string argv[])
{
    if (argc == 2)
    {
        printf("hello, %s\n", argv[1]);
    }
    else
    {
        printf("hello, world\n");
    }
}

اجرای ./greet David باعث می‌شود برنامه بگوید hello, David.

وضعیت خروج

وقتی یک برنامه پایان می‌یابد، یک کد خروج ارائه می‌شود:

#include 
#include 

int main(int argc, string argv[])
{
    if (argc != 2)
    {
        printf("Missing command-line argument\n");
        return 1;
    }
    printf("hello, %s\n", argv[1]);
    return 0;
}

می‌توانید echo $? را در ترمینال تایپ کنید تا وضعیت خروج را ببینید.

رمزنگاری

رمزنگاری هنر رمزگذاری و رمزگشایی یک پیام است. plaintext و یک key به یک cipher ارائه می‌شوند، که منجر به متن رمزگذاری‌شده می‌شود.

جمع‌بندی

در این درس، شما جزئیات بیشتری در مورد کامپایل و نحوه ذخیره داده‌ها در کامپیوتر یاد گرفتید. به طور خاص، شما آموختید:

  • نحوه کار یک کامپایلر.
  • چگونه کد خود را با استفاده از چهار روش اشکال‌زدایی کنید.
  • چگونه از آرایه‌ها در کد خود استفاده کنید.
  • چگونه آرایه‌ها داده‌ها را در بخش‌های پشت سر هم حافظه ذخیره می‌کنند.
  • چگونه رشته‌ها به سادگی آرایه‌هایی از کاراکترها هستند.
  • چگونه با آرایه‌ها در کد خود تعامل کنید.
  • چگونه آرگومان‌های خط فرمان می‌توانند به برنامه‌های شما منتقل شوند.
  • بلوک‌های سازنده اساسی رمزنگاری.
